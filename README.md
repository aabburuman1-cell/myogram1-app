# MyoGram - تطبيق شبيه بإنستغرام

## 📱 وصف المشروع
تطبيق موبايل حديث يشبه Instagram يوفر تجربة تفاعلية مع الريلز والمحادثات والميزات الاجتماعية.

## ✨ الميزات الرئيسية

### 1. **الصفحة الرئيسية (Home)**
- عرض الريلز (فيديوهات قصيرة)
- قصص المستخدمين
- تفاعل مع المنشورات (إعجاب، تعليق، مشاركة)
- ديزاين احترافي وسلس

### 2. **البحث (Search)**
- البحث عن مستخدمين آخرين
- عرض نتائج البحث مباشرة
- متابعة المستخدمين
- عرض معلومات المستخدمين

### 3. **الرسائل (Messages)**
- المحادثات الفردية
- واجهة محادثات سلسة
- عرض آخر الرسائل
- إشعارات الرسائل غير المقروءة

### 4. **الإشعارات (Notifications)**
- إشعارات الإعجاب
- إشعارات المتابعة
- إشعارات التعليقات
- تصنيف الإشعارات حسب النوع

### 5. **الملف الشخصي (Profile)**
- عرض معلومات المستخدم
- شبكة المنشورات
- إحصائيات المتابعين والمتابعة
- تعديل الملف الشخصي

### 6. **شاشة التسجيل (Login)**
- واجهة تسجيل دخول احترافية
- التحقق من البيانات
- عرض/إخفاء كلمة المرور
- خيار إنساء حساب جديد

## 🛠️ التقنيات المستخدمة

- **React Native** - إطار العمل الرئيسي
- **Expo** - منصة تطوير React Native
- **React Navigation** - نظام التنقل
- **Firebase** - قاعدة البيانات والمصادقة (إذا لزم الأمر)
- **React Native Vector Icons** - الأيقونات

## 📋 متطلبات التشغيل

```bash
# تثبيت Node.js و npm
# https://nodejs.org/

# تثبيت Expo CLI
npm install -g expo-cli

# تثبيت المتطلبات
npm install
```

## 🚀 كيفية تشغيل المشروع

```bash
# تشغيل التطبيق
npm start

# أو للويب
npm run web

# أو للأندرويد
npm run android

# أو للـ iOS
npm run ios
```

## 📁 هيكل المشروع

myogram1-app/
